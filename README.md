# Toys Store Data Analysis Project

## Project Overview
This project performs a comprehensive SQL-based analysis of the **Toys Store dataset** from [Kaggle](https://www.kaggle.com/datasets/mysarahmadbhat/toy-sales) to extract actionable insights on sales performance, product success, inventory management, and profitability.

---

## Tools & Technologies
- **SQL**: Data querying, aggregation, and analysis
- **Power BI**: Dashboards and data visualization
